# parejas-barajadas
Crea un programa que dado un array de personas, realice emparejamientos aleatorios.

Especificaciones:
	* El programa debe de sacar por pantalla cada pareja elegida.
	* El programa deberá de ser completamente aleatorio cada vez que se ejecuta.
	* El programa en caso de impares, deberá de añadir a la persona suelta a una pareja aleatoria. Formando así un trio.
